# 📌 Git vs GitHub – Quick Revision Notes

---

## 🔹 Git

- A **Distributed Version Control System (DVCS)**.
- Tracks changes in code locally.
- Works offline.
- Common commands:
  - `git init`
  - `git add`
  - `git commit`
  - `git branch`
  - `git merge`
- Git is a **software/tool** installed on your system.

---

## 🔹 GitHub

- A **cloud-based hosting platform** for Git repositories.
- Used for collaboration, Pull Requests (PRs), and code reviews.
- Provides:
  - UI for repositories
  - Issues tracking
  - GitHub Actions (CI/CD)
  - Access control & team management
- Requires internet connection.
- GitHub is a **service/platform**.

---

## 🧠 In One Line

👉 **Git** = Tool to track code changes  
👉 **GitHub** = Platform to store & collaborate on Git repositories
